Propriedades CSS

Centralização

A propriedade text-align em CSS pode ser usada para centralizar texto ou outros elementos dentro de seu contêiner.

Exemplo:

                    
                        .centralizado {
                            text-align: center;
                        }

Cor

A propriedade color em CSS define a cor do texto dentro de um elemento.

Exemplo:

                    
                        .texto-vermelho {
                            color: red;
                        }

Tamanho

A propriedade font-size em CSS define o tamanho da fonte de texto.

Exemplo:

                    
                        .grande {
                            font-size: 24px;
                        }
